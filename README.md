# Alma Labs - Agencia de IA Autónoma

## Visión
Crear una agencia de IA autónoma usando CrewAI y Coolify con sistema de memoria multi-nivel.

## Estructura
- **agents/**: Definiciones de agentes (CEO, CTO, Supervisor, Portero, etc.)
- **memory/**: Sistema de memoria común y privada con Qdrant + Cipher
- **infrastructure/**: Configuración Coolify, Docker, despliegues
- **workflows/**: Automatizaciones n8n
- **docs/**: Documentación del proyecto

## Stack Tecnológico
- CrewAI
- Qdrant
- Supabase  
- Cipher
- n8n
- Coolify
